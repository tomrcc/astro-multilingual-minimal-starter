Ce modèle utilise des fichiers de données pour :

* Alimenter les champs select et multiselect dans CloudCannon.
  * Couleurs
  * Icônes
  * Tags
* Définir des valeurs à l'échelle du site, telles que les paramètres SEO généraux.
  * Titre du site
  * Description
  * Image OG
  * Type de page
* Contrôler les données d'en-tête et de pied de page pour permettre aux éditeurs de gérer la navigation.
  * Logo
  * Couleurs
  * Liens de page
  * Liens sociaux
  * Droits d'auteur
